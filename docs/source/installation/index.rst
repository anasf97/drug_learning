Installation
====================

From Conda (recommended)
--------------------------

::

 conda create -n drug_learning_env ...

 conda activate drug_learning_env ...

 conda install ...


From PyPi
-----------

::

 pip install drug_learning


From Source Code
---------------------

::

 git clone https://github.com/anasf97/drug_learning

 cd drug_learning

 python setup.py install
